To get the api calls to work you need to do the following.

Setup Flask
    1. From a terminal cd into Flask folder
    2. Do a pipenv install
    3. Do a pipenv shell
    4. Run python app.py from within the pipenv shell
        - this launches my website at http://127.0.0.1:8000

Setup api
    1. From a new terminal cd into SWC Project
    2. Do a pipenv install
    3. Do a pipenv shell
    4. Run api_call.py from winthin the pipenv
